### Features
- User authentication with JWT
- Role-based access (client/counselor)
- Browse counselors, appointment booking
- Real-time chat
- Jitsi Meet for video calls (Chrome/Edge recommended)
- Payment flow (mock for demo)
- Mobile responsive

### How to run locally

#### Backend
cd backend
npm install
npm run dev
- Create `.env` as per `.env.example` (add MongoDB, JWT secret)

#### Frontend
cd frontend
npm install
npm start
- Add `.env` with your backend API URL
